Penjelasan dengan langkah-langkah:

• Pesawat memiliki 48 kursi

• 1 Penumpang kelas utama : bagasi 60 kg

• 1 Penumpang kelas ekonomi : bagasi 20 kg

• Pesawat dapat membawa : bagasi 1440 kg

• Harga tiket kelas utama : 150.000

• Harga tiket kelas ekonomi : 100.000

Kita buat persamaan Matematika,

misalkan,

- Kursi pesawat = Kp

- Penumpang kelas utama = Pku

- Penumpang kelas ekonomi = Pke

- Bagasi kelas utama = Bku

- Bagasi kelas ekononmi = Bku

- Jumlah bagasi Maksimum pesawat = Bpm

- Harga tiket kelas utama = Hku

- Harga tiket kelas ekonomi = Hke

• Kp = 48

• Pku + Pke = Kp

• Pku + Pke = 48 ... pers 1

• Bpm = 1440

• 60Bku + 20Bke = Bpm

• 60Bku + 20Bke = 1440

• 1 Pku = 60Bku

• 1 Pke = 20Bke

• 60Pku + 20Pke = 1440 ... pers 2

60Pku + 20Pke = 1440 ... pers 2• nPku (1.500.000) + nPke (1.000.000) = .... pers 3 (Hasil nilai yang optimum)

Untuk menetukan nilai Optimum, maka kita dapat menentukan nilai setiap nPku dan nPke

• Pku max, syarat Pke = 0

60Pku + 20Pke = 1440

60Pku = 1440

Pku = 24

HP = {nPku, nPke} = {24, 0} ... Hasil 1

• Pke max, syarat Pku = 0

60Pku + 20Pke = 1440

20Pke = 1440

Pku = 72

HP = {nPku, nPke} = {0, 72} ... Hasil 2

• Pke = 48,

Pku + Pke = 48

Pku + 48 = 48

Pku = 0

HP = {nPku, nPke} = {0, 48} ... Hasil 3

• eliminasi Pke dengan pers 1 dikali 20 dan pers 1 dikali 1

20Pku + 20Pke = 960

60Pku + 20Pke = 1440

__________________-

-40Pku = -480

Pku = 12

• Substitusi nilai Pku ke pers 1

Pku + Pke = 48

12 + Pke = 48

Pke = 36

HP = { nPku, nPke } = {12, 36} ... Hasil 4

Pada sebuah biro perjalanan pak Andri memesan 6 buah tiket pesawat A dan 2 buah tiket pesawat B dengan harga Rp6.100.000,00, sedangkan pak Ricky memesan 4 buah tiket pesawat A dan sebuah tiket pesawat B dengan harga Rp3.850.000,00 pada biro perjalanan yang sama. Harga sebuah tiket pesawat A dan harga sebuah tiket pesawat B berturut-turut adalah ….


Jawab:

Pada kasus Pak Andri:

6.A + 2.B = 6.100.000

Pada kasus Pak Ricky:

4.A + B = 3.850.000

Dengan metode eliminasi, diperoleh:

Pak Andri: 6A + 2B = 6.100.000

Pak Ricky (dikali 2): 8A + 2B = 7.700.000

sehingga -2A = -1.600.000

A = 800.000

Pada kasus Pak Ricky:

4A + B = 3.850.000

4 x 800.000 + B = 3.850.000

B = 650.000.

Jadi harga tiket pesawat A adalah Rp 800.000, harga tiket pesawat B adalah Rp 650.000.

banyaknya penumpang kelas utama : a
banyaknya penumpang kelas ekonomi: b

a + b <= 60  --> banyaknya kursi
60a + 20b = 1680 --> banyaknya bagasi

apabila semua penumpangnya kelas utama, maka kapasitas bagasi pesawat 1680 kg / 60 kg bagasi = 28 orang saja yang dapat ditampung

apabila semua penumpangnya kelas eknomi, maka kapasitas bagasi pesawat 1680 kg / 20 kg bagasi = 84 orang, tapi kapasitas kursi hanya 60 orang.

eliminasi
60 a + 20 b = 1680 
20 a + 20 b = 1200 -
-------------------------------
40 a             = 480

35. pesawat penumpang mempunyai tempat duduk 40 kursi. setiap penumpang kelas utama boleh membawa bagasi 60 kg sedangkan kelas ekonomi 20 kg. pesawat hanya dapat membawa bagasi 1.440 kg. harga tiket kelas utama Rp1.250.000,- & kelas ekonomi Rp750.000,-. agar pendapatan dari penjualan tiket pada saat pesawat penuh mencapai pendapatan maksimum, tiket yang terjual adalah??


fungsi objektif = f(x, y) = 1250000x + 750000y
kendala :
x + y ≤ 40
60x + 20y ≤ 1440

cari titik potong.

x + y = 40
x = 40 - y

60x + 20y = 1440
60(40 - y) + 20y = 1440
2400 - 60y + 20y = 1440
-40y = -960
40y = 960
y = 960/40
y = 24

x + y = 40
x + 24 = 40
x = 40 - 24
x = 16

titik pertama = (16, 24)

misal, x = 0
60x + 20y = 1440
20y = 1440
y = 72
namun karena maksimal kursi 40, jadi ambil y = 40

titik kedua = (0, 40)

misal y = 0
60x + 20y = 1440
60x = 1440
x = 1440/60
x = 24

titik ketiga = (24, 0)

jadi, didapatkan titik yang akan diuji :
f(16, 24)
f(0, 40)
f(24, 0)

masukan ke dalam fungsi objektif
f(16, 24) = 16(1250000) + 24(750000) = 38000000
f(0, 40) = 0(1250000) + 40(750000) = 30000000
f(24, 0) = 24(1250000) + 0(750000) = 30000000

Jadi, agar pendapatannya maksimum, tiket yang dijual adalah 16 tiket kelas utama dan 24 tiket kelas ekonomi.
