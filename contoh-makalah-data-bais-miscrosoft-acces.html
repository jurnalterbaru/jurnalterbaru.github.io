5. contoh Quick acces dan fungsinya​


Jawaban:

Quick Access adalah fitur yang memungkinkan pengguna untuk mengakses file atau folder yang sering dibuka dengan cepat dan mudah. Quick Access biasanya terletak di bagian atas jendela File Explorer atau Finder pada sistem operasi Windows dan Mac, masing-masing.

Contohnya, jika Anda sering mengakses folder "Document" untuk menyimpan file-file Anda, Anda dapat menambahkan folder tersebut ke Quick Access. Kemudian, Anda dapat dengan cepat mengakses folder tersebut dengan mengklik ikon folder di bagian atas jendela File Explorer atau Finder.

Fungsi Quick Access adalah untuk mempermudah dan mempercepat akses ke file atau folder yang sering digunakan. Ini juga memungkinkan pengguna untuk mengakses file atau folder tersebut dari berbagai perangkat yang terhubung ke akun yang sama, sehingga pengguna dapat dengan cepat mengakses file atau folder yang dibutuhkan dari mana saja.

10. jelaskan fungsi dari tipe data pada M.s Acces


Jawaban:

 

Fungsi dari tipe data pada MS Acces adalah:

 

1.    Text: tipe data ini berfungsi untuk menyimpan teks, angka atau kombinasinya. Ukuran yang dapat ditampung dalam tipe data ini maksimum adalah 255 karakter.         

2.    Memo: tipe data ini berfungsi untuk menyimpan teks, angka atau kombinasinya dan mirip seperti tipe data Text, namun tipe data Memo memiliki kapasitas lebih besar hingga 65,536 karakter. Meski lebih besar kapasitasnya, kelemahan tipe data Memo ini adalah tidak dapat disortir, namun masih dapat di-search (dicari isinya).

3.    Byte: untuk menyimpan angka dengan nilai antara 0 sampai 255.

4.    Integer: untuk menyimpan angka dengan nilai antara -32,768 dan 32,767.

5.    Long: untuk menyimpan angka dengan nilai antara -2,147,483,648 and 2,147,483,647.

6.    Single: untuk menyimpan data dengan jenis Single precision floating-point dengan ukuran 32 bit (4 byte)

7.    Double: untuk menyimpan data dengan jenis Double precision floating-point dengan ukuran 64 bit (8 byte)

8.    Currency: untuk menyimpan data, yang terdiri dari 15 digit nilai mata uang, umumnya dollar, dan 4 digit nilai desimal.

9.    AutoNumber: untuk menyimpan nilai yang diisi secara otomatis oleh database, dimulai dari angka 1.

10. Date/Time: untuk menyimpan yang terdiri dari tahun, bulan dan tanggal

11. Yes/No: untuk menyimpan nilai logika seperti ya/tidak, benar/salah, on/off. Tipe data ini tidak boleh kosong.

12. Ole Object: untuk menyimpan file berukuran besar seperti gambar, video dan musik, atau data BLOB (binary large object) lainya

13. Hyperlink: untuk menyimpan link ke halaman web.

14. Lookup Wizard: untuk menyimpan pilihan dalam bentuk dropdown list

20. type data pada program ms acces 2013


penjelasannya sebagai berikut :

1. Text

Text digunakan untuk field alfanumeric (misalnya nama, alamat, kode pos, dll), jadi tipe data text disini memiliki banyak karakter dan karakter tersebut ialah 255 karakter pada setiap fieldnya.

2. Memo

Tipe data memo sama halnya dengan text, tetapi tipe data memo dapat menampung kurang lebih 64.000 karakter untuk tiap fieldnya, tapi tidak bisa diurutkan/diindekskan.

3. Number

Tipe data Number dapat digunakan untuk menyimpan data numeric yang akan digunakan untuk proses perhitungan matematis (misalnya mengurangi, menambahkan, mengkali dan membagi suatu bilangan)

4. Date/Time

Tipe data Data/time digunakan untuk data yang berjenis tanggal, waktu atau penggabungan dari tanggal dan waktu

5. Currency

Tipe data Currency yaitu data dengan jenis number, hanya saja pada awal angka selalu disertakan symbol currency default sesuai dengan regional setting yang digunakan, misalnya RP. $, dll. Currency dapat menggunakan angka dengan 15 dgiit dibelakang desimal dan 4 digit sesudah desimal

6. AutoNumber

Tipe data Auto Number adalah data yang tidak dapat kita isi secara manual melainkan ia terisi secara otomatis oleh Access, baik secara menjumlah ataupun random (acak) , AutoNumber biasanya digunakan untuk penomoran.

7. YES/NO

Tipe data Yes/No merupakan tipe data dengan 2 pilihan saja yaitu Yes (1 atau Ture) dan No (0 atau False). dan format yang tersedia adalah Yes/No, True/False, dan On/OFF

8. OLE Object

Tipe data OLE Object digunakan untuk eksternal objek, seperti bitmap atau file suara

9. Hyperlink

Tipe data Hyperlink digunakan untuk menyimpan alamat internet atau file yang ditunjukan melalui alamat URL

10. Lookup Wizard

Jika menggunakan tipe data ini untuk sebuah field, maka bisa memilih sebuah nilai dari tabel lain atau dari sebuah daftar nilai yang ditampilkan dalam combobox

11. Attachment

Attatchment itu merupakan tipe data yang digunakan untuk melampirkan sebuah data apapun bentuk filenya / ekstensinya.

30. Suku kedua suatu baisan geometri adalah 72 dan suku keempatnya adalah 8 .Rasio dari baisan tesebut adalah positif . Suku ketujuh dai baisan tesebut adalah….


kelas : XII SMA
mapel : matematika
kategori : baris dan deret
kata kunci : barisan geometri

kode : 12.2.7 [matematika SMA kelas 12 Bab 7 baris dan deret]

Pembahasan:

Barisan geometri adalah barisan bilangan yang mempunyai rasio tetap antara dua suku barisan yang berurutan.
selisih antar suku barisan geometri disebut rasio (dilambangkan dengan r). Artinya, suku barisan ditentukan oleh perkalian atau pembagian oleh suatu bilangan tetap dari suku barisan sebelumnya.

rumus umum barisan geometri adalah
Un = [tex]a. r^{n-1} [/tex]

dalam soal diketahui:
U2 = 72
U4 = 8
r > 0
ditanya U7 dari barisan tersebut adalah ...
jawab:

U4 = a.r³ = 8
U2 = a.r  = 72
-------------------- :
           r² = 1/9
           r = √(1/9)
             = 1/3

subsitusikan nilai r ke U2
U2 = a.r = 72
         a. (1/3) = 72
         a = 72 x 3
         a = 216

U7 = [tex] a. r^{7-1} [/tex]
      = 216. [tex] (\frac{1}{3} )^{6} [/tex]
      = 216 . 1/729
      = 216/729
      = 24/81
      = 8/27

jadi suku ke tujuh dari barisan tersebut adalah 8/27

35. sejarah miscrosoft di indonesia?


Microsoft didirikan untuk mengembangkan dan menjual penerjemah BASIC untuk Altair 8800. Perusahan ini berhasil mendominasi pasar sistem operasi komputer pribadidengan MS-DOS-nya pada pertengahan 1980-an, diikuti dengan jajaran sistem operasiMicrosoft Windows. Penawaran umum perdana Microsoft tahun 1986, dan kenaikan tajam harga sahamnya, menciptakan tiga miliuner dan 12.000 jutawan di kalangan karyawan Microsoft. Sejak 1990-an, perusahaan ini semakin terdiversifikasi dari pasar sistem operasi dan telah melakukan sejumlah akuisisi perusahaan.
