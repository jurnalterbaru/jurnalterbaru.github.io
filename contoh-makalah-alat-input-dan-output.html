10. apa yang dimaksud dengan alat input ,alat proses dan alat output


Alat input adalah alat yang berada di dalam CPU tepatnya di motherboard seperti Ram,dan lain-lain.Sedangkan alat proses merupakan aplikasi atau layanan yang dipasang untuk membantu jalannya sistem operasi.Alat output merupakan alat yang digunakan untuk menyempurnakan aplikasi input,seperti Mouse,monitor,keyboardkalo alat input adalah alat alat yang befungsi untuk memasukan data ke dalam komputer.
kalo alat proses adalah alat alat yang berfungsi untuk mengolah data ke dalam komputer setelah proses input.
kalo alat output adalah alat alat yang berfungsi untuk mengeluarkan data data yang berbentuk informasi.

26. QUIZ!!!Sebutkan pengertian dari input, proses, output serta contoh alat-alatnya! note : informatika ini ​


Jawaban:

Alat Input adalah alat-alat yang berfungsi memasukan data atau perintah kedalam komputer. contoh keyboard, mouse, scanner

Alat Output adalah alat-alat yang berfungsi mengeluarkan data-data yang berbentuk informasi. contoh monitor, speaker, printer

Alat Proses adalah alat-alat yang berfungsi mengolah data kedalam komputer setelah mengalami proses Input.Contoh dari alat proses adalah Processor
