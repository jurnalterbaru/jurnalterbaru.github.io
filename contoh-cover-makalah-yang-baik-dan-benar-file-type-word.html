4. Bagaimana cara membuat daftar isi pada makalah dengan menggunakan word sesuai kaidah bahasa Indonesyang baik dan benar


Cara Membuat Daftar Isi

Berikut ini cara membuat daftar isi yang baik, karena dengan daftar isi yang baik maka pembaca akan merasa puas.

Pertama, dalam membuat daftar isi maka kita harus menguasai materi yang ada pada buku atau laporan tersebut. Maka dari itu sebaiknya membaca secara keseluruhan, akan lebih bagus lagi jika kita sendiri yang menulis buku atau laporan itu sehingga kita pasti tau seluk beluk yang terkandung didalamnya.

Kedua, untuk membuat daftar isi yang baik maka harus mencakup semua materi melalui poin penting. Untuk itu catat terlebih dahulu BAB, Sub-BAB, dan poin yang ada didalam sub bab tersebut.

Ketiga, setelah catatan tersebut dirasa lengkap maka kamu harus mengetahui jenis yang akan ditulis seperti apa. Karena daftar isi
kliping resep masakan dengan daftar isi proposal atau daftar isi laporan itu berbeda, biasanya yang sedikit resmi ada bagian Prakata atau pembuka, pendahuluan. Tapi jika laporan tugas misalkan untuk kliping saja, biasanya langsung ke bab.

Format Open XML dan ekstensi nama file

Excel 2013 Word 2013 PowerPoint 2013 Excel 2010 Word 2010 Selengkapnya...

Mulai dari sistem Microsoft Office 2007, Microsoft Office menggunakan format file berbasis XML, seperti .docx, .xlsx, dan .pptx. Apa saja manfaat dari Format Open XML?

Format Open XML menyertakan banyak manfaat — bukan hanya bagi pengembang dan solusi yang mereka susun, tetapi juga bagi perorangan dan organisasi dalam segala ukuran:

  File ringkas     File dipadatkan secara otomatis dan pada beberapa kasus, dapat menjadi 75 persen lebih kecil. Format Open XML menggunakan teknologi pemadatan zip untuk menyimpan dokumen, menawarkan potensi penghematan biaya karena mengurangi ruang disk yang dibutuhkan untuk menyimpan file dan mengurangi bandwidth yang diperlukan untuk mengirimkan file melalui email, lewat jaringan, dan melintasi internet. Bila Anda membuka file, maka akan dimekarkan secara otomatis. Bila Anda menyimpan file, maka akan dipadatkan lagi secara otomatis. Anda tidak perlu menginstal utilitas zip khusus untuk membuka dan menutup file dalam Office.

  Pemulihan file rusak yang telah disempurnakan     File disusun secara modular yang menyimpan berbagai komponen data berbeda dalam file yang terpisah satu sama lain. Hal ini memungkinkan file dibuka bahkan jika komponen dalam file (misalnya, bagan atau tabel) rusak.

 Privasi dan kontrol yang lebih baik terhadap informasi pribadi     Dokumen dapat dibagikan secara rahasia, karena informasi yang bisa diidentifikasi secara pribadi dan informasi rahasia bisnis, seperti nama penulis, komentar, perubahan terlacak, dan jalur file dapat diidentifikasi dan dihapus dengan mudah dengan menggunakan Pemeriksa Dokumen.

 Integrasi dan interoperabilitas data bisnis yang lebih baik     Menggunakan Format Open XML sebagai kerangka kerja interoperabilitas data untuk rangkaian produk Office berarti bahwa dokumen, lembar kerja, presentasi, dan formulir dapat disimpan dalam format file XML yang tersedia secara gratis untuk digunakan atau dilisensikan oleh siapa saja, bebas royalti. Office juga mendukung skema XML yang ditentukan-pelanggan yang menyempurnakan tipe dokumen Office yang ada. Ini berarti bahwa pelanggan dapat dengan mudah membuka kunci informasi dalam sistem yang ada dan melakukan tindakan sesuai informasi tersebut dalam program Office biasa. Informasi yang dibuat dalam Office dapat digunakan dengan mudah oleh aplikasi bisnis lain. Yang Anda butuhkan untuk membuka dan mengedit file Office hanyalah utilitas ZIP dan editor XML.

 Deteksi dokumen yang memuat makro menjadi lebih mudah     File yang disimpan dengan menggunakan akhiran "x" default (seperti .docx, .xlsx, dan .pptx) tidak mungkin memuat makro Visual Basic for Applications (VBA) dan makro XLM. Hanya file yang ekstensi nama filenya berakhiran "m" (seperti .docm, .xlsm, dan .pptm) yang dapat memuat makro.

Atas Halaman

Apa saja ekstensi nama file XML?

Secara default, dokumen, lembar kerja, dan presentasi yang Anda buat dalam Office disimpan dalam format XML dengan ekstensi nama file yang menambahkan huruf "x" atau "m" pada ekstensi nama file yang biasa Anda kenal. Huruf "x" berarti file XML yang tidak memiliki makro, dan "m" berarti file XML yang memuat makro. Misalnya, bila Anda menyimpan dokumen dalam Word, sekarang file menggunakan ekstensi nama file .docx secara default, bukan lagi ekstensi nama file .doc.

Menyimpan file sebagai .docx

Bila Anda menyimpan file sebagai templat, Anda akan mendapati perubahan yang sama. Ekstensi templat yang digunakan dalam versi lama masih ada, tetapi sekarang diakhiri dengan huruf "x" atau "m". Bila file memuat kode atau makro, Anda harus menyimpannya dengan menggunakan format file XML dengan makro aktif yang baru, yang ditambahi dengan huruf "m", kependekan dari makro pada ekstensi file.

23. Bagaimana contoh penulisan daftar pustaka yang baik dan benar dalam makalah ?


Jawaban:

1. Nama

Nama penulis ditulis paling awal. Ingatlah untuk selalu menuliskan nama belakang penulis terlebih dahulu, kemudian dilanjutkan dengan tanda koma (,) setelah menulis nama depan dan tengah penulis buku tersebut. Jika buku tersebut merupakan karya dari dua penulis atau lebih, hanya penulis pertama yang urutan namanya dibalik. Penulis kedua dan seterusnya setelahnya dengan urutan yang sesuai dengan nama lagu tersebut. Jika pada buku tersebut nama penulis dicantumkan lengkap dengan gelar pendidikan atau gelar lain, gelar-gelar tersebut tidak perlu dituliskan.

2. Tahun Terbit

Setelah nama, terbitkan tahun terbit dari buku yang teman-teman gunakan sebagai referensi. Jangan terkecoh pada angka tahun beton awal sebab bisa saja buku yang kamu pakai merupakan beton kedua, ketiga, ataupun terakhir.

3. Judul Buku

judul judul bukumu secara lengkap. Jangan lupa, penulisan judul dibuat dengan italic ( miring ).

4. Kota dan Nama Penerbit

Bagian terakhir dalam penulisan daftar pustaka sebuah buku adalah penerbitan buku tersebut dan nama penerbit yang mencetak. Dahulukan penulisan nama kota, baru diikuti dengan nama penerbit yang dibatasi dengan tanda titik dua (:).
