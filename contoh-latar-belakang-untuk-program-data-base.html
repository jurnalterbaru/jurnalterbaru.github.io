17. Contoh Dalam Membuat Data Base


contoh misalkan ada tabel mahasiswa,tabel ini berisi tentang data data mahasiswa

contoh nya terdapat kolom

NPM        nama_mahasiswa        kode_prodi       semester      kelas      alamat

di dalam database ada tabel dan di dalam tabel ada kolom, bukan hanya tabel yang berada di database juga ada biasanya trigger,fungsi,view,dll

Berikut adalah contoh database yang dibuat dengan menggunakan MS SQL Server:

CREATE DATABASE SimpleLibrary

GO

USE SimpleLibrary

GO

CREATE TABLE msUser

(

Username VARCHAR(100),

Password VARCHAR(100),

BukuDiPinjam INT,

AuditUsername CHAR

)

INSERT INTO msUser (Username, Password, BukuDiPinjam, AuditUsername)

VALUES ('admin','admin', 0,'D')

-- =============================================

-- Author: Pyrozen  

-- Create date: 18 February 2019

-- Description: Login validation for Simple Library

-- =============================================

--bn_SimpleLibrary_DoSignIn

CREATE PROCEDURE bn_SimpleLibrary_DoSignIn

@Username VARCHAR(500),@Password VARCHAR(500)

AS

BEGIN

SELECT Username,Password,BukuDiPinjam,AuditUsername

FROM msUser

WHERE Username = @Username And Password = @Password  

 And AuditUsername<>'D'

END

EXEC bn_SimpleLibrary_DoSignIn 'admin','admin'

Pembahasan

Pertama, kita membuat database yang bernama SimpleLibrary dan langsung menggunakannya. Kemudian kita membuat sebuah tabel di database tersebut yang bernama msUser atau master data user dimana berisi kolom Username, Password, BukuDiPinjam, dan AuditUsername. Setelah itu kita memasukkan value baris pertama untuk admin. Terakhir adalah optional, jika ingin digunakan pada framework lain seperti ASP .NET maka kita membuat Stored Procedure. Stored procedure yang dibuat digunakan untuk validasi login User dengan cara mencari data yang sesuai, jika tidak ada maka akan return null.

Penjelasan:

CV atau data diri penting untuk pendaftaran seminar proposal karena saat seminar nanti data dirimu akan dijelaskan oleh sang moderator misal seperti tempat tanggal lahir, riwayat pendidikanmu, ataupun pengalaman lainnya

Beberapa universitas meminta kamu untuk menunjukan bukti kehadiran seminar yang telah kamu hadiri sebelumnya agar kamu bisa seminar. Hal ini semata-mata dilakukan supaya kamu punya pengalaman nanti seminar seperti apa dengan menonton orang-orang yang sebelumnya telah seminar proposal
